# Carmel De-vries · Homemade Baked Goods

דף נחיתה ומחירונים לעסק האפייה הביתית של קארמל די-וריס.

## תוכן הריפו

```
├── index.html               ← דף הנחיתה הראשי (עם הזמנה לווטסאפ)
├── menu-1-classic.html      ← מחירון, גרסה אלגנטית קלאסית
├── menu-2-luxury.html       ← מחירון, גרסה דרמטית כהה
├── menu-3-modern.html       ← מחירון, גרסה מודרנית בכרטיסים
├── images/                  ← תמונות המוצרים
└── README.md                ← הקובץ הזה
```

## פריסה ל-GitHub Pages (פעם אחת)

1. בריפו → לשונית **Settings** (למעלה)
2. בתפריט הצד שמאל → **Pages**
3. תחת **Branch** בוחרים: `main` + תיקייה: `/ (root)` → **Save**
4. ממתינים 1-2 דקות, מקבלים ירוק עם הקישור: `https://USERNAME.github.io/carmel-devries/`

זהו. כל push ל-`main` יעדכן את האתר אוטומטית.
